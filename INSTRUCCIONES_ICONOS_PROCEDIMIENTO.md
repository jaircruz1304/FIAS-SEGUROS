# Cambio: íconos visuales en Procedimiento rápido

Reemplaza estos archivos en tu repositorio:

```text
seguro-vehicular.html
assets/css/styles.css
assets/js/app.js
```

El ajuste corrige que se visualicen palabras técnicas como `shield`, `phone`, `camera`, `ban`, `alert` y `file`.

Ahora el sistema convierte esos valores del Excel/JSON en íconos SVG visuales:

- `shield` = seguridad
- `phone` = llamada a asistencia
- `camera` = registro fotográfico/video
- `ban` = no pactar acuerdos
- `alert` = no pagos/reparaciones sin autorización
- `file` = comunicar/documentar a FIAS

No debes cambiar el Excel ni los JSON. Puedes mantener esos códigos en la columna `icono`.
