# Mejora compacta y coberturas

Reemplaza estos archivos en el repositorio:

- `seguro-vehicular.html`
- `assets/css/styles.css`
- `assets/js/app.js`
- `data/seguros/seguros-vehiculares.json`
- `data/seguros/coberturas.json`

Si la web se actualiza desde SharePoint/OneDrive, sube también el Excel actualizado a la ruta del Excel maestro:

- `templates/Base_Datos_Seguros_Vehiculares_FIAS.xlsx`

## Mejoras incluidas

- Se agregó una sección de "Daños y coberturas principales".
- Se usan paneles colapsables para reducir la extensión visual de la página.
- Las pólizas, bienes, videos, documentos y preguntas frecuentes quedan en secciones desplegables.
- Los bienes asegurados muestran solo el resumen; el detalle se abre cuando el usuario lo necesita.
- El procedimiento rápido ahora se muestra como pasos desplegables.
